# Smart AI Glass System

## Project Overview
Smart AI Glass is an intelligent system for real-time object detection, text recognition (OCR), scene summarization, and voice output. It is designed to run on lightweight devices like Raspberry Pi, providing users with a wearable smart-glass prototype.

## Features
- Real-time object detection (person, chair, door, etc.)
- Text recognition from camera frames (OCR)
- Scene summarization in natural language
- Voice-driven interface (Text-to-Speech using pyttsx3)
- Interaction logging for system events and results

## Requirements
- Python 3.9+
- Pillow
- opencv-python
- pytesseract
- pyttsx3

Install dependencies with:

pip install -r requirements.txt

## How to Run
1. Clone or download this repository.
2. Make sure all dependencies are installed.
3. Run the main system:

python smart_glass_system.py

4. Output snapshots, text logs, and TTS files will be saved in the `smart_glass_demo_output/` folder.
